Encrypted Messaging App

We developed an Encrypted Messaging App that focuses on providing secure communication through advanced encryption techniques. The application allows users to send and receive messages while ensuring their privacy and confidentiality.

Key Features

  • Encryption Algorithms: Implemented robust encryption methods using AES (Advanced Encryption Standard) for symmetric encryption and RSA (Rivest-Shamir-Adleman) for asymmetric encryption, ensuring that messages remain secure during transmission.
  • User-Friendly Interface: Designed a simple and intuitive graphical user interface (GUI) using PyGUI, enabling users to easily send and receive encrypted messages without technical complications.
  • Research on Messaging Services: Conducted thorough research on various messaging services provided by organizations to understand their encryption practices and improve upon them.

Technologies Used

  • Python for application development.
  • PyGUI for building the graphical user interface.
  • Cryptography libraries for AES and RSA encryption.
