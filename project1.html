Cloud Computing Chatbot

We developed a sophisticated chatbot focused on cloud computing using the RASA framework. This project aims to assist users by providing quick and accurate answers to their queries related to cloud technologies, services, and best practices.

Key Features

  • Natural Language Processing: Utilized RASA's NLP capabilities to understand user queries and provide relevant responses.
  • Integration: Deployed the chatbot in a cloud environment to allow users to access it anytime, enhancing the user experience.
  • Deployment on Google Colab: Developed and tested the chatbot in Google Colab, leveraging its powerful computing resources for efficient model training and evaluation.

Technologies Used

  • RASA
  • Python
  • Google Colab
